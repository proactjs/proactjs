Class: Observable

ProAct. Observable

new Observable(transforms)

Constructs a ProAct.Observable. It can be used both as observer and observable.

The observables in ProAct.js form the dependency graph. If some observable listens to changes from another - it depends on it.

The observables can transform the values or events incoming to them.

Every observable can have a parent observable, that will be notified for all the changes on the child-observable, it is something as special observer.

ProAct.Observable is part of the core module of ProAct.js.

into() → {ProAct.Observable}

Links source observables into this observable. This means that this observable is listening for changes from the sources.

A good example is one stream to have another as as source -> if data comes into the source stream, it is passed to the listening too. That way the source stream is plugged into the listening one.

The listeners from ProAct.Observable#makeListener and ProAct.Observable#makeErrListener are used.

transform(transformation) → {ProAct.Observable}

Adds a new transformation to the list of transformations of this observable.

A transformation is a function or an object that has a call method defined. This function or call method should have one argument and to return a transformed version of it. If the returned value is ProAct.Observable.BadValue, the next transformations are skipped and the updating value/event becomes - bad value.

Every value/event that updates this observable will be transformed using the new transformation.

willUpdate(source, actions, eventData) → {ProAct.Observable}

Will update is the method used to notify observers that this ProAct.Observable will be updated.

It uses the ProAct.Observable#defer to defer the listeners of the listening ProAct.Observables. The idea is that everything should be executed in a running ProAct.Flow, so there will be no repetative updates.

The update value will come from the ProAct.Observable#makeEvent method and the source parameter will be passed to it.

If this ProAct.Observable has a parent ProAct.Observable it will be notified in the running flow as well.

Parameters:
Name Type Description
source Object The source of the update, for example update of ProAct.Observable, that this is observing.

Can be null - no source.

In the most cases ProAct.Event is the source.

actions Array | String A list of actions or a single action to update the listeners that listen to it. If there is no action provided, the actions from ProAct.Observable#defaultActions are used.
eventData Array Data to be passed to the event to be created.
