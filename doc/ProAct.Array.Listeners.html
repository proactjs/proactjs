<static, constant> every

Generates a listener that can be attached to an ProAct.Array on which the method ProAct.Array#pevery is invoked.

The result of the ProAct.Array#pevery method is a ProAct.Val, dependent on the original array.

For example if the original was: 

   var a = new ProAct.Array([1, 3, 5]);
and we invoked ProAct.Array#pevery on it like this: 
   var val = a.pevery(function (el) {
     return el % 2 === 1;
   }); // val.v is true.
The new value - val is dependent on a, so if for example we push something to a, val should be updated: 
   a.push(2); // val.v authomatically should become false.

The generated listener by this method does this - updates the val ProAct.Val, when the original array changes and it does it in an optimal way.
